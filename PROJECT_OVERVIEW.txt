================================================================================
  1. GIỚI THIỆU DỰ ÁN
================================================================================

Tên:        PreOnic (Công ty TNHH PreOnic Việt Nam)
Slogan:     Nền tảng kết nối nông nghiệp bền vững hàng đầu Việt Nam

Mô tả:      PreOnic là nền tảng số hóa chuỗi cung ứng nông sản, kết nối
            trực tiếp giữa Nông dân (Farmer) và Doanh nghiệp (Enterprise)
            trong mô hình bao tiêu nông sản. Nền tảng hỗ trợ toàn bộ vòng
            đời hợp đồng: từ tìm kiếm nguồn cung, ký kết hợp đồng điện tử,
            thanh toán escrow an toàn đến theo dõi giao hàng.

Mô hình:    - Nông dân đăng bán nông sản, xem hợp đồng, nhận thanh toán
            - Doanh nghiệp tìm nguồn cung, ký hợp đồng bao tiêu, quản lý kho
            - PreOnic thu phí hoa hồng 3% trên giá trị mỗi hợp đồng
            - Escrow: PreOnic giữ tiền cọc, giải ngân theo milestone

Tech Stack:
  Frontend:  React 19 + React Router v7 + React Bootstrap 5.3
             Framer Motion (animation) | recharts 3.7 (biểu đồ)
             Axios (HTTP) | Context API (state)
  Backend:   Node.js + Express.js + TypeScript
             MongoDB + Mongoose (database)
             JWT (access token 7d + refresh token 30d + httpOnly cookie)
             nodemailer (email) | multer/base64 (upload) | node-cron (cron)
  Tools:     OpenWeatherMap API | PayOS (cổng thanh toán — chưa tích hợp)
             Google OAuth (đã cấu hình .env — chưa implement)

Cấu trúc thư mục:
  e:\K7\exe\PreOnic\
  ├── be/          Backend Node.js + TypeScript (port 5000)
  └── fe/          Frontend React (port 3000)


================================================================================
  2. TỔNG QUAN CÁC CHỨC NĂNG CHÍNH
================================================================================

  A. XÁC THỰC & TÀI KHOẢN
     ─────────────────────
     - Đăng ký (farmer / enterprise), đăng nhập bằng email hoặc SĐT
     - JWT: access token (bearer) + refresh token (httpOnly cookie)
     - Auto-refresh token khi hết hạn (axios interceptor)
     - Brute-force protection: khóa 15 phút sau 5 lần sai mật khẩu
     - Quên mật khẩu / Đặt lại mật khẩu (token, BE đã có, email chưa gửi)
     - Xem & sửa hồ sơ cá nhân (/profile)
     - Soft-delete tài khoản (isActive = false)

  B. TRANG CHỦ & ĐIỀU HƯỚNG CÔNG KHAI
     ────────────────────────────────
     - Trang chủ (/) gồm: Hero, CompanyIntro, ProductsByRegion, Process,
       Campaigns, AgricultureBanner, Stats, Benefits, Solutions, Contact, Footer
     - /products   — Danh sách tất cả sản phẩm (lọc, tìm kiếm, sắp xếp)
     - /products/:id — Chi tiết sản phẩm (countdown, progress bar, modal cam kết)
     - /solutions  — Trang giải pháp (static)
     - /contact    — Trang liên hệ

  C. DASHBOARD NÔNG DÂN (/farmer)
     ─────────────────────────────
     - Tổng quan mùa vụ: thống kê cây trồng, hợp đồng, escrow balance (API)
     - Quản lý hợp đồng bao tiêu (dữ liệu từ DB)
     - Theo dõi đơn hàng
     - Tài chính: thu nhập, lịch sử giao dịch từ Escrow
     - Danh bạ đối tác (doanh nghiệp đã hợp tác)
     - Thị trường: giá nông sản, tin tức (hiện đang hardcode)
     - Tab Thời tiết: thời tiết hiện tại + dự báo 5 ngày (API)
     - Đăng bán nông sản mới (form 4 bước + upload ảnh/chứng chỉ)
     - Xác nhận milestone escrow, tạo tranh chấp

  D. DASHBOARD DOANH NGHIỆP (/enterprise)
     ──────────────────────────────────────
     - Tổng quan: stats hợp đồng, nhà cung cấp, số dư (API)
     - Quản lý hợp đồng bao tiêu
     - Danh sách sản phẩm cần thu mua
     - Theo dõi đơn hàng
     - Danh bạ nhà cung cấp (nông dân đã hợp tác)
     - Quản lý kho bãi
     - Phân tích thị trường (BarChart + LineChart — recharts, API)
     - Tab Thời tiết: thời tiết hiện tại + dự báo 5 ngày (API)
     - Nạp tiền escrow, xác nhận milestone, tạo tranh chấp

  E. QUY TRÌNH TẠO HỢP ĐỒNG (/contract-flow)
     ──────────────────────────────────────────
     - Luồng 5 bước: Đề xuất → Xem xét → PreOnic xác nhận → Ký → Hoàn tất
     - Tự tính tổng giá trị + phí hoa hồng 3%
     - Auto-fill từ URL ?product=<id>
     - Lưu hợp đồng vào MongoDB, trả về số hợp đồng PRE-YYYY-XXXX thực tế
     - Toggle bảo hiểm mùa vụ

  F. HỆ THỐNG ESCROW
     ─────────────────
     - Tạo escrow khi ký hợp đồng
     - Doanh nghiệp nạp tiền cọc (ghi DB, chưa có cổng thanh toán thực)
     - Xác nhận milestone từ cả hai phía → tự động giải ngân theo %
     - Tạo tranh chấp (dispute) với lý do

  G. NHẮN TIN (/messaging)
     ──────────────────────
     - Danh sách hội thoại (sidebar), khung chat (right panel)
     - REST API đầy đủ (conversations + messages)
     - Polling 5 giây tự động, optimistic send
     - Không real-time (không có WebSocket)

  H. THÔNG BÁO (NotificationBell)
     ───────────────────────────
     - Badge số thông báo chưa đọc trên header cả hai dashboard
     - Dropdown 10 thông báo gần nhất, mark as read
     - Cron job cảnh báo thời tiết mỗi 6 giờ

  I. UPLOAD FILE
     ────────────
     - Upload ảnh sản phẩm (tối thiểu 3 ảnh) + chứng chỉ (PDF/JPG)
     - Encode base64 → gửi lên BE → lưu vào /uploads/ trên disk
     - Preview ảnh trước khi gửi

  J. THỜI TIẾT
     ──────────
     - Lấy dữ liệu từ OpenWeatherMap API
     - Thời tiết hiện tại, dự báo 5 ngày, cảnh báo thời tiết
     - Cron job chạy tự động mỗi 6 giờ
